# Linear Algebra for ML — Code Implementations

Each folder contains a self-contained Python implementation of the concept covered in the slide deck.

## Structure

```
linear-algebra-for-ml/
├── 01_vectors/               vectors, norms, normalization, cosine similarity
├── 02_matrices/              matrix ops, matmul, transpose, broadcasting
├── 03_dot_products/          dot product, neuron simulation, cosine similarity
├── 04_eigenvalues/           eigendecomposition, PCA from scratch
├── 05_svd/                   SVD, image compression, LSA, recommender systems
├── 06_embeddings/            word embeddings, cosine similarity, king-man+woman
├── 07_transformers_attention/ scaled dot-product attention from scratch
```

## Requirements

```bash
pip install numpy matplotlib scikit-learn
```

Run any file directly:
```bash
python 01_vectors/vectors.py
```
